Step 1 : Install NodeModules of both client and server
Step 2 : run client by running npm start
Step 3 : run server by running npm start
Step 4 : Navigate to Frontend by going localhost:3000
Step 5 : After running the backend you will se console when you reloaded the app saying user connected
Step 6 : now Open postman and hit different routes and see the action being called without any refresh on the frontend 


The Routes are as follows: 

http://localhost:8888/socket
http://localhost:8888/socket/info
http://localhost:8888/socket/error
http://localhost:8888/socket/success
